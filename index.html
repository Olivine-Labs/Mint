Speed, flexibility, simplicity. Pick three.

Development done right.

Mint's built for modern web applications. Built by leading front-end and server-side engineers, it offers just the right amount of help; you won't get bogged down in configuration changes or be forced to learn any esoteric templating languages. By default, it ships using MongoDB and Mustache templating.

Scalable architecture.

As you build your web application, you're building a RESTful API. Design and expose your models by creating database models, and expose those collections as a JSON API. You consume your own API as you build your pages, and can trivially use Backbone.js or any client-side framework to build a javascript-driven application. It's your choice; Mint makes it easy.

Getting started is easy.

Even if you don't know Lua, it's very easy to pick up, especially if coming from a background in Javascript, Ruby, or any functional language. Drop Mint in your application as a Git subtree, or download the repository and plop the folder in your application's root. (We'll be releasing it as a Luarocks module in the future.)
