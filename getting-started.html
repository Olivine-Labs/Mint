Explanation of Setup

These instructions are based on what we found to be easeiest to work with. You may very well find it better to use a different hosting provider, or nginx instead of apache, or a different database solution; Mint doesn't have any restrictions, that's up to you. We set up defaults that accomodate rapid, modern development and allow you to change out modules externally to the Mint core systems.
